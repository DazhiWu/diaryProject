# 日记项目年度总结页面设计

## 设计思路

基于现有的日记项目架构，我将设计一个丰富有意义的年度总结页面，允许用户手动录入数据并通过可视化图表展示年度日记统计和反思。

## 实现计划

### 1. 创建年度总结页面组件
- 新建文件：`components/yearly-summary.tsx`
- 包含年度选择器、数据录入表单和可视化展示

### 2. 设计数据录入表单
- 日记数量统计（按月份）
- 情感分布（积极、中性、消极）
- 主题标签统计
- 重要事件记录
- 年度关键词
- 个人成长与反思

### 3. 实现数据可视化
- 使用 Recharts 绘制月度日记数量柱状图
- 情感分布饼图
- 主题标签云图
- 年度事件时间线

### 4. 页面布局设计
- 顶部：年度选择器和数据管理按钮
- 中部：多维度图表展示区
- 底部：文字总结和反思区

### 5. 集成到主应用
- 在 `app/page.tsx` 中添加年度总结视图选项
- 添加相应的路由和导航

### 6. 数据存储方案
- 使用 localStorage 临时存储用户录入的总结数据
- 提供导出/导入功能，方便用户保存和恢复数据

## 技术栈

- Next.js 13+（客户端组件）
- React Hooks（useState, useEffect）
- Recharts（数据可视化）
- Tailwind CSS（样式）
- 现有的 UI 组件库（卡片、表单、按钮等）

## 预期效果

- 用户可以选择年份并手动录入年度总结数据
- 通过直观的图表展示年度日记统计
- 提供美观的文字总结区域
- 支持数据的保存和导出
- 与现有项目风格保持一致
