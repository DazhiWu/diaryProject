# AI读后感双形态展示功能实现计划

## 功能需求
为AI读后感模块实现双形态展示，两个形态可以切换：
1. **第一形态**：原本的卡片展示（已存在）
2. **第二形态**：大屏展示形态，包含：
   - 中央显示选中的AI分析内容
   - 下方按钮根据卡片标题填充
   - 点击按钮更新中央文字为对应卡片的详情内容

## 实现步骤

### 1. 添加形态切换状态
在 `yearly-summary.tsx` 组件中添加状态管理，用于控制当前显示的形态：
```typescript
const [analysisViewMode, setAnalysisViewMode] = useState<'card' | 'screen'>('card')
```

### 2. 添加形态切换按钮
在AI读后感栏目标题右侧添加切换按钮组，用于在两种形态间切换：
```typescript
<div className="flex bg-muted p-1 rounded-lg">
  <Button
    variant={analysisViewMode === 'card' ? 'default' : 'ghost'}
    size="sm"
    onClick={() => setAnalysisViewMode('card')}
    className="h-8 px-3"
  >
    卡片视图
  </Button>
  <Button
    variant={analysisViewMode === 'screen' ? 'default' : 'ghost'}
    size="sm"
    onClick={() => setAnalysisViewMode('screen')}
    className="h-8 px-3"
  >
    大屏视图
  </Button>
</div>
```

### 3. 实现大屏展示形态
在AI读后感栏目中，根据 `analysisViewMode` 状态渲染不同的UI：

#### 3.1 大屏形态容器
创建一个全屏风格的容器，包含：
- 顶部标题
- 中央内容区域
- 底部按钮组

#### 3.2 中央内容区域
显示当前选中的AI分析内容，包括：
- 标题
- 主要内容
- 相关观点

#### 3.3 底部按钮组
根据AI分析列表动态生成按钮，点击按钮切换中央显示的内容：
```typescript
<div className="flex flex-wrap gap-2 justify-center mt-6">
  {yearlySummary.aiAnalyses.map((analysis) => (
    <Button
      key={analysis.id}
      variant={selectedAnalysisId === analysis.id ? 'default' : 'outline'}
      onClick={() => setSelectedAnalysisId(analysis.id)}
      className="px-4 py-2"
    >
      {analysis.title}
    </Button>
  ))}
</div>
```

### 4. 添加选中分析状态
添加状态管理当前选中的AI分析内容：
```typescript
const [selectedAnalysisId, setSelectedAnalysisId] = useState<string | number | null>(null)
```

### 5. 初始化选中状态
在数据加载完成后，默认选中第一个AI分析内容：
```typescript
useEffect(() => {
  if (yearlySummary.aiAnalyses.length > 0) {
    setSelectedAnalysisId(yearlySummary.aiAnalyses[0].id)
  }
}, [yearlySummary.aiAnalyses])
```

### 6. 样式设计
为大屏形态添加现代化的样式，包括：
- 渐变背景
- 圆角边框
- 响应式设计
- 平滑过渡动画

## 代码修改位置

1. **组件状态**：在 `YearlySummary` 组件顶部添加新的状态变量
2. **UI渲染**：在AI读后感栏目中添加条件渲染逻辑
3. **切换按钮**：在AI读后感标题右侧添加视图切换按钮
4. **大屏形态UI**：实现新的大屏展示形态组件

## 预期效果

- 用户可以通过按钮在两种形态间切换
- 大屏形态下，中央显示选中的AI分析内容
- 点击底部按钮可以切换不同的AI分析内容
- 两种形态的切换平滑自然

## 技术要点

- 使用React的条件渲染实现形态切换
- 利用状态管理控制当前选中的内容
- 动态生成按钮组
- 现代化CSS样式设计
- 响应式布局适配不同屏幕尺寸
